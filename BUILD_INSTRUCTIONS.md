# Build Instructions

## Prerequisites
- Node.js 18+ and npm

## Build Steps
1. Install dependencies:
```
   npm install
```

2. Build the extension:
```
   npm run build:firefox
```

3. Output will be in `.output/firefox-mv2/`

## Tools
- Uses WXT framework (https://wxt.dev)
- Built with Vite and TypeScript
- No external APIs or services used
